# Elevation

Shadows designed for visual hierarchy and animating interactive elements.

https://material.io/design/environment/elevation.html

## Animation

Elevation can be animated nicely:

<DisplayRectangle
  color={theme.colors.brand.white}
  shadow={theme.elevation['01dp']}
  hover={theme.elevation['16dp']}
  active={theme.elevation['08dp']}
  name="normal: elevation['01dp'], hover: elevation['16dp'], active: elevation['08dp']"
/>

## Levels

Elevation levels are accessed through the theme, e.g. `theme.elevation['06dp']`.
